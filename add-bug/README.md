# add-bug

Log a bug report to `docs/bugs.md` in the current project, with timestamp, description, and status tracking.

## Usage

```
/add-bug
```

Then describe the bug. The skill will append a structured entry to `docs/bugs.md` (created if missing).

## Pairs well with

- [`list-bugs`](../list-bugs) — view what's logged.
- [`fix-bug`](../fix-bug) — work through them.
